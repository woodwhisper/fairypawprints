This site

  • This site could be coded as a simple website, but is instead a more complex 'single' page app using AJAX.
  • Using an old version of twitter bootstrap at the moment for basic layout and navigation.
  • Using jQuery for Ajax and back end page load.
  • Using Angular for page automation
  • The contact us page is designed to be as anti spam as possible without adding extra user stuff, plus feeds to a PHP form to be processed
  • The backend is a API PHP system using MVC techniques with Doctrine 2 as it's Database layer.
